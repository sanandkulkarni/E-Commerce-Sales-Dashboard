# E-Commerce Sales Analysis

## Overview
Analyzed e-commerce sales data using Power BI and DAX to understand business performance, customer behavior, and sales trends.

## Tools Used
- Power BI, DAX Excel

## Key Analysis
- Performance Overview: Tracked Total Sales, Profit, and Quantity
- Geographical Analysis: Identified top revenue-generating states
- Product Analysis: Found high-performing categories and profit-driving sub-categories
- Customer Insights: Analyzed top customers and revenue contribution
- Payment Analysis: Evaluated preferred payment methods
- Time Trends: Identified monthly trends and seasonal patterns

## Dashboard Features
- KPI Cards (Sales, Profit, Quantity)
- Region-wise analysis
- Category & product performance
- Customer insights
- Payment distribution
- Time-series trends
- Interactive slicers

## Key Insights
- Top states contribute majority of revenue
- Few product categories drive most profits
- High-value customers generate significant sales
- Sales show seasonal fluctuations
